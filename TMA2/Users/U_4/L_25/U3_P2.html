So now that we understands some basics of how to write XML, lets look at how we can use XML in our webpage. The Practice of applying Asynchronous JavaScript And XML to a webpage is often refered to as AJAX. This technique makes use of the XMLHttpRequest object in order to manipulate what is refered to as the HTML DOM. A Document Object Model(DOM) is created by the browser when the webpage is loaded, and it defines all the content within the HTML. To JavaScript, the DOM acts as an API to allow Javascript to manipulate it. Let's take a look at our html file real quick:
